### Docker Compose: 
[Tell about how to update the Docker files and container.]


Updating maven and components: 
navigate into "Gift_Basket_Production" folder. 

Run clean install. 
```bash
mvn clean install
```


Run the Docker-Compose and start the container. Remember to start Docker Desktop. 
```bash
docker-compose up
```


Connect to the running container. 
```bash
docker exec -it 4semesterproject-sem4-1 bash
```

Run the Project from inside the Container. 
```bash
mvn exec:exec
```
